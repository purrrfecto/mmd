## 必要情報

如果有興趣的話請提供以下資訊：
1. 一個公開的推特帳號 （非公開的話無法加進列表)
2. 一個你發表的MMD影片或圖片連結
    - 必須上傳在公開的網站niconico/youtube/twitter等，有遵守各種規章**並能確認你是本人**（例如有寫了你申請用的推特帳號之類的）
3. 一個你喜歡的彩虹社EN實況影片或剪片烤肉等
4. 你下載用的bowroll帳號連結
